# Xyz Demo 项目概述

## 项目简介

Xyz Demo 是一个基于 Spring AI 框架构建的多智能体系统演示项目，集成了大语言模型（LLM）、MCP 协议、Milvus 向量数据库等前沿技术，展示了企业级 AI 应用的最佳实践。

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Java | 17 | LTS 版本 |
| Spring Boot | 3.5.10 | 核心框架 |
| Spring AI | 1.1.2 | AI 集成框架 |
| Spring AI Alibaba | 1.1.2.0 | 阿里云 AI 扩展 |
| Spring AI Alibaba Agent | 1.1.2.2 | 智能体框架 |
| AgentScope | 1.0.10 | 多智能体编排 |
| Milvus | 2.6.12 | 向量数据库 |
| MySQL | 8.3.0 | 关系型数据库 |
| MyBatis | 3.5.15 | ORM 框架 |

## 快速开始

### 环境要求

- JDK 17+
- Maven 3.8+
- MySQL 8.0+
- Milvus 2.0+（可选，用于向量检索）

### 构建命令

```bash
# 编译项目
mvn clean compile -s D:\soft\apache-maven-3.8.1\conf\settings.xml

# 打包项目
mvn clean package -DskipTests

# 运行项目
java -jar ai-demo/target/ai-demo-0.0.1-SNAPSHOT.jar
```

## 核心功能

1. **多智能体编排** - 支持顺序、并行、路由等多种编排模式
2. **RAG 检索增强** - 基于 Milvus 的知识库检索
3. **MCP 协议集成** - 支持模型上下文协议
4. **工具调用** - 支持 Function Calling
5. **流程编排** - 支持复杂工作流定义
