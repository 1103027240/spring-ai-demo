# 工具调用（Function Calling）

## 概述

工具调用允许 LLM 在生成响应时调用外部函数，实现与外部系统的交互。本项目基于 Spring AI 实现了完整的工具调用机制。

## 工具定义

### 函数工具位置

- `ai-demo/function/` - AI 模块的函数工具
- `agent-demo/function/` - Agent 模块的函数工具

### 内置工具

| 工具类 | 功能 |
|--------|------|
| `WeatherFunction` | 天气查询 |
| `SqlExecuteFunction` | SQL 执行 |
| `DemoToolCallbackProvider` | 示例工具提供者 |

## 工具定义示例

```java
@Component
public class WeatherFunction implements Supplier<FunctionCallback> {

    @Override
    public FunctionCallback get() {
        return FunctionCallback.builder()
            .name("get_weather")
            .description("获取指定城市的天气信息")
            .inputType(WeatherRequest.class)
            .function(this::execute)
            .build();
    }
    
    private WeatherResponse execute(WeatherRequest request) {
        // 实现天气查询逻辑
        return new WeatherResponse(request.getCity(), "晴天", 25);
    }
}
```

## 工具注册

在配置类中注册工具：

```java
@Configuration
public class ToolConfig {
    
    @Bean
    public ToolCallbackProvider toolCallbackProvider(
            WeatherFunction weatherFunction,
            SqlExecuteFunction sqlFunction) {
        return new DefaultToolCallbackProvider(
            weatherFunction.get(),
            sqlFunction.get()
        );
    }
}
```

## 工具调用流程

```
1. 用户发送问题
       ↓
2. LLM 分析是否需要调用工具
       ↓
3. 如果需要，生成工具调用请求
       ↓
4. 执行工具函数
       ↓
5. 将结果返回给 LLM
       ↓
6. LLM 生成最终响应
```
