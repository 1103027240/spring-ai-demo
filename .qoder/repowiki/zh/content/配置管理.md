# 配置管理

## 概述

项目采用 Spring Boot 标准配置方式，支持多环境配置和外部化配置。

## 配置文件位置

| 文件 | 模块 | 说明 |
|------|------|------|
| `ai-demo/src/main/resources/application.yml` | ai-demo | 主配置文件 |
| `agent-demo/src/main/resources/application.yml` | agent-demo | Agent 配置 |
| `mcp-client-demo/src/main/resources/application.yml` | mcp-client | MCP 客户端配置 |
| `mcp-server-demo/src/main/resources/application.yml` | mcp-server | MCP 服务端配置 |

## 核心配置项

### LLM 配置

```yaml
spring:
  ai:
    dashscope:  # 阿里云通义千问
      api-key: ${DASHSCOPE_API_KEY}
      chat:
        options:
          model: qwen-plus
          temperature: 0.7
```

### Milvus 配置

```yaml
milvus:
  host: localhost
  port: 19530
  database: default
```

### MySQL 配置

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/spring_ai_demo
    username: root
    password: ${MYSQL_PASSWORD}
    driver-class-name: com.mysql.cj.jdbc.Driver
```

### Redis 配置

```yaml
spring:
  data:
    redis:
      host: localhost
      port: 6379
      password: ${REDIS_PASSWORD}
```

## 配置类

### AI 模块配置

| 配置类 | 功能 |
|--------|------|
| `SaaLLMConfig` | LLM 模型配置 |
| `MilvusCollectionCreatorConfig` | Milvus 集合配置 |
| `MysqlSaverConfig` | MySQL 存储配置 |
| `RedisConfig` | Redis 缓存配置 |
| `ObjectMapperConfig` | JSON 序列化配置 |

### Agent 模块配置

| 配置类 | 功能 |
|--------|------|
| `CustomAgentLoaderConfig` | 自定义智能体加载 |
| `SwaggerConfig` | API 文档配置 |

## 环境变量

| 变量名 | 说明 |
|--------|------|
| `DASHSCOPE_API_KEY` | 阿里云 API 密钥 |
| `MYSQL_PASSWORD` | MySQL 密码 |
| `REDIS_PASSWORD` | Redis 密码 |

```bash
mvn clean compile -s D:\soft\apache-maven-3.8.1\conf\settings.xml
```
