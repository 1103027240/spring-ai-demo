# 多智能体系统

## 概述

本项目基于 Spring AI Alibaba Agent 和 AgentScope 构建了完整的多智能体系统，支持多种编排模式。

## 智能体类型

### 1. 顺序编排（Sequential）

按顺序依次执行多个智能体，前一个智能体的输出作为后一个的输入。

```
智能体A → 智能体B → 智能体C
```

**配置位置**: `agent-demo/config/sequential/`

### 2. 并行编排（Parallel）

多个智能体同时执行，最后合并结果。

```
         ┌→ 智能体A ─┐
输入 ──→ ├→ 智能体B ─┼──→ 结果合并
         └→ 智能体C ─┘
```

**配置位置**: `agent-demo/config/parallel/`

### 3. 路由编排（Routing）

根据输入内容动态路由到不同的智能体处理。

```
         ┌→ 智能体A (条件1)
输入 ──→ 路由器 ─┼→ 智能体B (条件2)
         └→ 智能体C (条件3)
```

**配置位置**: `agent-demo/config/routing/`

### 4. 循环编排（Loop）

智能体循环执行直到满足终止条件。

```
┌──────────────────┐
│                  ↓
输入 ──→ 智能体 ──→ 条件判断 ──→ 输出
         ↑              │
         └──────────────┘
```

**配置位置**: `agent-demo/config/loop/`

## 核心组件

### 智能体构建器

位于 `agent-demo/build/` 目录：

| 文件 | 说明 |
|------|------|
| `ChatMessageBuild.java` | 聊天消息构建 |
| `GraphBuild.java` | 流程图构建 |
| `MultiAgentBuild.java` | 多智能体构建 |
| `ToolCallbackBuild.java` | 工具回调构建 |
| `CodeSearchRoutingBuild.java` | 代码搜索路由构建 |

### 智能体服务

位于 `agent-demo/service/` 目录，提供各类智能体服务实现。

## 控制器接口

| 控制器 | 功能 |
|--------|------|
| `MultiSequentialController` | 顺序编排接口 |
| `MultiParallelController` | 并行编排接口 |
| `MultiRoutingController` | 路由编排接口 |
| `MultiLoopController` | 循环编排接口 |
| `MultiA2AController` | Agent-to-Agent 通信接口 |

## 使用示例

```java
// 注入智能体服务
@Autowired
private MultiAgentService multiAgentService;

// 执行顺序编排
public String executeSequential(String input) {
    return multiAgentService.executeSequential(input);
}
```
